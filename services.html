Services

In order to use any of these services, you will need to authenticate either with a BrowserID account or a local Ubuntu account, depending on the type of service. Please be aware that every single service here is broken, for various reasons.

SSH

SSH requires a command-line shell (such as bash), an SSH client (like ssh), and an account on the local Ubuntu system (to request an account, email me). If you can't connect on Windows, I recommend PuTTY if you don't feel like setting anything up. Otherwise, I recommend Cygwin. It's way more than just an SSH client, it provides a *nix-like environment on Windows.

In case this link doesn't work, you can connect to alexj.jumpingcrab.com, usually by running the command ssh [username]@alexj.jumpingcrab.com in a terminal, replacing [username] with your username.

FTP

FTP will only provide access to your home directory. FTP requires an FTP client (one may be included in your browser) and an account on the local Ubuntu system (to request an account, email me).

In case this link doesn't work, just connect to alexj.jumpingcrab.com in your FTP client.
