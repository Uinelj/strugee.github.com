Services

All services here will eventually be backed by an LDAP directory, but for now, they're kind of all hacked together with different account backends.

SSH

SSH requires an SSH client and an account on the local Debian system (to request an account, email hostmaster).

In case this link doesn't work, you can connect to strugee.net as usual on port 22 (ssh [username]@strugee.net). (This page used to recommend you connect to ssh.strugee.net, but that is no longer necessary as of the move away from GitHub Pages.)

Mail

I run all my personal mail off strugee.net. If you're having problems with the mail services running on strugee.net, contact postmaster.

ownCloud

I run an instance of ownCloud at this location.

MediaWiki

wiki.strugee.net currently runs an instance of MediaWiki. It's used to document server configuration. You can find the exact details of services on strugee.net there, as well as less important services not listed here.
