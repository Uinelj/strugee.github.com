Services

In order to use any of these services, you will need to authenticate either with a BrowserID account or a local Ubuntu account, depending on the type of service.

Emergence Story Project

The Emergence Story Project requires a modern browser (Firefox and Chrome are the best) and a BrowserID account. You can also look at an exact list of browser requirements if you want.

The Emergence Story Project is currently in an incomplete form and is not functional. When it is done, it will run a check to make sure that your browser has the necessary capabilities to run it. If your browser is not supported, you will have to upgrade it - the Story Project depends on modern HTML5 features like WebSockets in order to run. For advanced users, the Story Project will provide a list of test faliures so that you can see exactly where your browser falls short.

SSH

SSH requires a command-line shell (such as bash), an SSH client (like ssh), and an account on the local Ubuntu system (to request an account, email me).

In case this link doesn't work, you can connect to alexj.jumpingcrab.com, usually by running the command ssh [username]@alexj.jumpingcrab.com in a terminal, replacing [username] with your username.

FTP

FTP will only provide access to your home directory. FTP requires an FTP client (one may be included in your browser) and an account on the local Ubuntu system (to request an account, email me).

In case this link doesn't work, just connect to alexj.jumpingcrab.com in your FTP client.
